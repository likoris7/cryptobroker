# NexTrade: Crypto Business Broker MVP
> Premium Turnkey Brokerage Platform

Вітаємо з придбанням! Цей проект повністю готовий до роботи. Він використовує Telegram Bot API як безкоштовну серверну інфраструктуру (Serverless CRM) для генерації лідів.

## Швидкий старт (Запуск за 5 хвилин)

Щоб сайт почав надсилати заявки у ВАШ Telegram, вам потрібно змінити лише 2 рядки коду у файлі `app.js`.

### Крок 1. Створіть власного бота
1. Відкрийте Telegram і знайдіть офіційного бота **@BotFather**.
2. Відправте команду `/newbot` і створіть свого бота (дайте йому ім'я, наприклад `MyBrokerBot`).
3. BotFather видасть вам **Bot Token** (довгий рядок символів). Скопіюйте його.
4. Обов'язково знайдіть свого новоствореного бота в пошуку Telegram і натисніть **"Start"** (Розпочати), щоб він міг надсилати вам повідомлення.

### Крок 2. Дізнайтесь свій Chat ID
1. У Telegram знайдіть бота **@getmyid_bot**.
2. Натисніть "Start".
3. Скопіюйте цифри з поля `Your user ID:` (наприклад `123456789`).

### Крок 3. Підключіть бота до сайту
1. Відкрийте файл `app.js` у будь-якому текстовому редакторі.
2. Знайдіть рядки коду (приблизно рядок 114):
```javascript
const BOT_TOKEN = "ВАШ_ТОКЕН_З_КРОКУ_1";
const CHAT_ID = "ВАШ_ID_З_КРОКУ_2";
```
3. Вставте туди свої дані замість моїх та збережіть файл.

### Крок 4. Публікація в Інтернет (Deploy)
Завантажте всі файли на ваш GitHub-акаунт. 
Після цього зайдіть на [Vercel.com](https://vercel.com) або використовуйте GitHub Pages:
- На Vercel натисніть "Add New Project".
- Виберіть ваш репозиторій.
- Натисніть "Deploy".
Через 15 секунд ваш сайт буде доступний на весь світ!
